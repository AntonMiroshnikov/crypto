    Описание проекта.
Данная программа реализует сдвиговый шифр и шифр простой замены. Программа состоит из шести модулей: главный модуль (main), модуль абстрактного класса (Cipher), модули классов  сдвигового шифра (ShiftCipher) и шифра простой замены (SubstitutionCipher), производных от класса Cipher, модуль класса исключений (CryptoExceptions), модуль функций для проверки входных данных (CheckInput).
